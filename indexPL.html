Cześć, nazywam się Sebastian Nowak.

Projektuję i tworzę aplikacje internetowe.

Jestem studentem geodezji zajmującym się tworzeniem baz danych. Specjalizuję się również w językach programowania takich jak HTML, CSS, JS.

Obecnie uczesniczę w BootCampie tworzonym przez Frontend Masters by rozwinąć się w programowaniu typu FullStack.

Kursy które ukończyłem

Najnowszy kurs

Getting Started with CSS

Kurs polegał na zagadnieniach dotyczących tworzenia funkcjonalnej stony internetowej z wykorzystaniem językow HTML oraz CSS. Nauczyłem się na nim metod i wskazówek zawewniających estetyczny wygląd strony. Kurs był prowadzony przez Jen Kramer na platformie Frontend Masters.

Technologie użyte w tym kursie:

  • HTML
  • CSS

Complete Intro to Web Development, v2

Ten kurs był przygotowaniem do tworzenia aplikacji webowych. Nauczyłem się na nim pisania kodów w trzech językach: HTML (tworzącym trzon skryptu), CSS (odpowiedzialnym za styl i design strony) oraz JS (wprowadzający funkcjonalność). Kurs był prowadzony przez Briana Holt na platformieFrontend Masters.

Technologie użyte w tym kursie:

  • HTML
  • CSS
  • Java Script

Automatyzacja testów: Selenium WebDriver + Java

Kurs przygotowywał do tworzenia testów automatycznych to testowania stron internetowych. Testowałem funkcjonalność aplikacji poprzez auto-uzupełnianie formularzy oraz korzystanie z funkcji na stronie przy użyciu skryptów pisanych w języku Java. Korzystałem z narzędzi takich jak: Selenium WebDriver, IntelliJ, JUnit, Maven, DevTools. Kurs był prowadzony przez Krzysztofa Nowakowskiego na platformie Udemy.

Technologie użyte w tym kursie:

  • Java
